# Fongdong-s-blinky-project
A led chaser guided project

Hardware :

1)NE555P Timer

2)4017 IC

3)2 Resistors(1k ohm and 470 ohm)

4)1 Potentiometer

5)1 Normal Capacitor

6)1 Polarized Capacitor

7)10 LEDs
